# Software & Access Requests

## Self-Service vs Ticket

| You want to...                                  | How to get it                                                |
| ----------------------------------------------- | ------------------------------------------------------------ |
| Install Slack, Zoom, Chrome, common dev tools   | Open **Self Service** on your laptop                         |
| Get a license for paid software (Figma, Linear) | File a ticket with your manager CC'd                         |
| Get added to a Slack channel                     | Ask the channel owner directly - IT can't add you            |
| Access to a shared Google Drive                  | Ask the drive owner; IT only manages org-wide drives         |
| Access to internal tools (Jira, Datadog)         | File a ticket; manager approval is automatic via Okta groups |
| Production database access                       | File a ticket; requires VP-level approval and security review |

## How Approvals Work

Most access requests run through Okta workflows:

1. You submit a request (ticket or Okta self-service request).
2. Your manager gets an approval email - they have 3 business days to approve.
3. Okta provisions access automatically once approved.
4. You'll get a Slack DM from `@okta-bot` confirming.

## Removing Access

- Access is automatically removed when you change teams or leave the company.
- To proactively give up access (e.g. you no longer need a license), file a ticket - this saves the company money on per-seat licenses.
