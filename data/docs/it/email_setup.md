# Email & Calendar Setup

Trumid uses Google Workspace. Your email is `firstname.lastname@trumid.com`.

## Mobile (iOS / Android)

1. Install the **Gmail** app and the **Google Calendar** app.
2. Add an account → Google → sign in with your Trumid email.
3. Approve the Okta Verify push.
4. Allow the device management profile when prompted - this lets IT remote-wipe the corporate account only if your phone is lost.

## Desktop (Outlook on macOS / Windows)

If you prefer Outlook:

1. Open Outlook → Add Account → enter your Trumid email.
2. Outlook will redirect to Okta - sign in there.
3. Choose **IMAP** if asked (not POP).

## Calendar Sharing

- By default, your free/busy is visible to anyone at Trumid.
- To share full event details with a specific person, open Calendar → Settings → Add people.
- For team-wide calendars (e.g. on-call rotations), ask your manager to file a ticket so IT can create a resource calendar.

## Aliases & Distribution Lists

- To request a new distribution list or alias (`team-name@trumid.com`), file a ticket with the requested address and intended members.
- You cannot self-serve aliases - they require security review.
